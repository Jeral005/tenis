Eventos en java script

javaScript puede cambiar esta imagen

Un evento es una acción que ocurrre sobre un elemento html, por ejemplo "click", "cuando la pagina carga", "cuando el usuario sale de la pagina", "cuando el cursor del raton pasa por sobre una imagen" o "cuando el cursor del raton esta sobre una imagen y el usuario lo aleja de la misma".
